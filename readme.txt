Upload `adsense-plugin` folder to the `/wp-content/plugins/` directory.2. Activate the plugin through the 'Plugins' menu in WordPress.3. Site wide Settings are located in "BWS Plugin", "AdSense".== Frequently Asked Questions === How to change Publisher ID =Go to the Settings page and change value for the 'Publisher ID' field.= How to change position of the ad block =At the page of plugin settings open menu item 'Position & Numbers of Ads' and on the list 'Position' choose appropriate for you position.== Screenshots ==1.
